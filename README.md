# Computer-Vision
Програма розпізнає об’єкти у реальному часі за допомогою моделі YOLOv5. Виявлені об’єкти перекладаються на українську, підраховується їхня кількість та записується лог у файл detections.txt. На екрані відображаються рамки навколо об’єктів з назвами та відсотком впевненості, а також топ-3 найбільш численних категорій.
Для запуску потрібні Python та бібліотеки ultralytics і opencv-python. Програму можна запустити просто з редактору кнопкою RUN.
Після запуску включається вебкамера, на відео видно виявлені об’єкти, їхні підрахунки та логування у файл для подальшого аналізу.
